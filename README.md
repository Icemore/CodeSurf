CodeSurf
========

Визуализация исполнения программ. При помощи GDB и чего-то еще.

Вдохновлялись:

* Демосцена
* AudioSurf
* Trip videos

Requirements:

* python 2.7
* panda3d 1.8.1
* gdb
* numpy
* scipy
* рекомендуется дискретная видеокарта и optirun

Как запустить:

* cd bin | g++ main.cpp -g
* python start.py, рекомендуется использовать дискретную видеокарту (optirun python start.py)
